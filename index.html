Reciprocal function: 1/x^2

1. Reciprocal function - ( 1.4 / x + 0.69 ) - 4.85 { -0.54 < x < 5.05 } asymptote x = 4
The function 1/x has been transformed into ( 1.4 / x + 0.69 ) - 4.85. The graph’s first transformation is a vertical stretch by the factor of 1.4 in the numerator stretching the graph vertically, making it steeper than the parent function 1/x . The second transformation is a shift to the left by 0.69 units, caused by the ( x + 0.69 ) in the denominator to fit the roller coaster. The last transformation is a vertical shift downwards by 4.85, moving the entire function to a lower coordinate plane..

Reciprocal square function

1. Transformations Paragraph
The function 1/x^2 has been transformed into 1.0( 34.7/x^2+2 ) - 5 . The graph’s first transformation is a vertical stretch by the factor of 34.7 in the numerator stretching the graph vertically and increasing the steepness of the graph. By adding 2 to the denominator it makes the function values smaller for all the x, causing a vertical compression of the graph without shifting it horizontally to fit the curve of the roller coaster. The last transformation is a vertical shift downwards by 5 shifting to a lower on the coordinate plane. The combined transformations result in a reciprocal square graph that remains its characteristic U-shaped hyperbolic form

Sinusoidal function

1. Transformations Paragraph
3sin (x/2.4 ) + 1 {0 < x < 8.7} Transformation - The parent function y=sin(x) was chosen as the roller coaster had smooth curves rising. I transformed the parent function into 3sin(x/2.4)+1 as 3 causes a vertical stretch, making the peaks taller to match the height of the hill. The division by 2.4 inside the sin creates horizontal stretch, which stretches the width of the waves out so it fits the roller coaster track. Adding +1 shifts the whole function upwards by 1 unit, so the track sits above the baseline rather than centered around it.

Exponential function

1. Transformations Paragraph
0.9(0.45^x) - 6 {-2.6 < x < 0} asymptote x = 1.2 y = -6 Transformation - The parent function y=a^x has a rapid decrease and flattening shape of exponential decay, which matches the roller coaster but it is transformed to 0.9(0.45^x)-6 to better fit a steep drop that levels out. As x increases, the base 0.45 makes the graph decrease. I vertically compressed the function by factor 0.9 to make the drop less extreme and more fitting to the roller coaster's exact height. I subtracted 6 to shift the entire graph downward by 6 units, allowing the curve to match the elevation of the track. This also creates the horizontal asymptote at y=-6, showing that the track flattens out.

domain {-2.6 < x < 0} range {-5.07 < x < 1.1}

Quadratic function

1. Transformations Paragraph
-0.3x^2 + 1.3 { -3.63 < x < 3.76 } Transformation - The function f(x) = 0.3x^2 + 1.3 is a transformation from the parent function y = x^2 . There is a vertical stretch by the factor 0.3, which decreases the steepness of the parabola and makes it wider compared to the parent function. The last transformation of the graph is a vertical shift upwards by 1.3 which raises the vertex of the parabola from the origin (0,0) to the point (0,1.3) . The transformed function transforms to a wider parabola that is elevated above the x-axis while retaining the symmetric U-shaped form.

domain { -3.63 < x < 3.76 } range { -2.91 < x < 1.3 }
