Reciprocal Function: 1/x²

1. Reciprocal Function - (1.4 / (x + 0.69)) - 4.85

The graph’s first transformation is a vertical stretch by the factor of 1.4 in the numerator stretching the graph vertically, making it steeper than the parent function 1/x. The second transformation is a shift to the left by 0.69 units caused by (x + 0.69) in the denominator. The last transformation is a vertical shift downwards by 4.85, moving the entire function lower on the coordinate plane.

2. Domain and Range

Domain: { -0.54 < x < 5.05 }
Range: { −4.61 < x < 4 }
