Reciprocal Function: 1/x²

Reciprocal Function

The function (1.4 / x + 0.69) - 4.85 for −0.54 < x < 5.05 has an asymptote at x = 4.
Transformation - The function 1/x has been transformed into (1.4 / x + 0.69) - 4.85. It’s vertically stretched by a factor of 1.4, shifted left by 0.69 units, and moved down by 4.85.

Domain and Range

Domain: { −0.54 < x < 5.05 }
Range: { −4.61 < y < 4 }

Reciprocal Square Function

Transformations

The function (34.7/x² + 2) - 5 is transformed from 1/x². It’s vertically stretched by 34.7, compressed vertically by +2 in the denominator, and shifted down by 5.

Domain and Range

Domain: { −1.916 ≥ x ≥ −7.4 }, { 1.6 < x < 7 }
Range: { −4.16 < y < 1.93 }, { −4.3 < y < 2.4 }

Polynomial Function (Degree > 2)

Transformations

Domain and Range
